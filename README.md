# HSalsa20.NET

A .NET implementation of [HSalsa20](https://cr.yp.to/snuffle/xsalsa-20110204.pdf).

> **Warning**
>
> HSalsa20 is **NOT** a hash function and **MUST NOT** be used to derive keys from passwords.
